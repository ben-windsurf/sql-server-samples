About This Analysis

This dashboard displays 12-month revenue and margin trends from the WideWorldImportersDW database. The data is processed and stored in Supabase for real-time visualization and is fully compatible with PowerBI and Kyvos for advanced analytics.

Key Metrics:

  • Total Revenue: Sum of all sales including tax
  • Total Margin: Profit from all transactions
  • Margin Percentage: Profit margin as percentage of revenue
  • Growth Rates: Month-over-month percentage changes

Data Sources:

  • Fact.Sale table (transaction data)
  • Dimension.Date table (time hierarchy)
  • Rolling 12-month window for trend analysis
